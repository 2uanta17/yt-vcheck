## The Problem

As someone who listens to music on Youtube, the fucking platform keeps region-locking the videos and reuploading them. Those region-locked videos do not have an indicator when you click "show unavailable video" so they're only known when you click on them.

## The Tech Stack

This web app uses:

- Angular (with Signals, because why not)
- Angular Material 3
- Angular CDK Virtual Scroll
- YouTube Data API v3
- .NET 9 Minimal API (it's just sitting in the folder lol)

You may ask: why is this techstack? I don't know, I thought I might want to learn Minimal APIs and Angular so I vibecoded using it, in the end I learned nothing though.

## What it does

- It identifies the "ghost" videos that are deleted, private, or region-locked.
- It detects duplicates so you can throw the extra ones in the trash.
- Direct links to find exactly which "Deleted video" ghost to nuke in your playlist.
- It checks things in batches of 50 so it's not super slow.

## Performance and Logic

Since I didn't want to pay for a server, I refactored everything into the frontend.

- **Batching:** It queries 50 items at a time because that's the API limit and it keeps things moving.
- **Virtual Scroll:** Even if you have 900+ items, the browser won't explode because we only render what's on screen.
- **Serverless:** It's 100% on GitHub Pages.

## Privacy

Everything stays in your browser. I have no database, no server, and I'm not stealing your API keys. Your data is your business.

One thing though: you do need your own YouTube Data API v3 key to actually run the scan.

## Getting Started

If you want to run this locally for some reason:

```bash
git clone https://github.com/2uanta17/yt-vcheck.git
cd yt-vcheck/frontend-deploy
npm install
ng serve
```

Go to `http://localhost:4200/`, paste your API key, and find out what's missing from your playlists.
